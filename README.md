# 🤖 Multimodal Chatbot Using Gemini Flash

A multimodal AI chatbot built with **Google Gemini 1.5 Flash** and **Streamlit** that enables users to interact using both **text** and **images**. The chatbot generates fast, context-aware responses by leveraging Gemini's multimodal capabilities.

## 🚀 Features

- 💬 Text-based conversations
- 🖼️ Image upload and analysis
- ⚡ Fast responses powered by Gemini 1.5 Flash
- 🎨 Interactive Streamlit web interface
- 🔒 Secure Google API key input

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini 1.5 Flash API
- Pillow (PIL)

## 📂 Project Structure

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/multimodal-chatbot-gemini-flash.git
cd multimodal-chatbot-gemini-flash
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run gemini_multimodal_chatbot.py
```

4. Enter your **Google Gemini API Key** when prompted.

## 📸 How It Works

1. Enter your Google Gemini API key.
2. Upload an image (optional).
3. Ask a question related to the image or provide a text prompt.
4. Receive intelligent, multimodal responses generated by Gemini Flash.
